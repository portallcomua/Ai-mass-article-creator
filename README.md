# AI Mass Article Creator

**Generate 1-20 unique SEO articles with AI, automatic images, and 3-5 REAL user comments per article.**

## 🚀 Features

| Feature | Description |
|---------|-------------|
| **Bulk Generation** | 1 to 20 articles with one click |
| **Real Comments** | 3-5 realistic user comments per article |
| **Smart Titles** | AI generates unique titles automatically |
| **Thematic Images** | Featured + inline images from Pexels/Unsplash |
| **SEO Ready** | Meta keywords & description |
| **Auto Tags** | Relevant tags extracted from content |
| **Uniqueness Check** | One-click verification on Quetext |
| **Bilingual** | Ukrainian & English support |
| **Auto Updates** | GitHub-based updates |

## 🔧 Installation

1. Upload `ai-mass-article-creator` to `/wp-content/plugins/`
2. Activate the plugin
3. Get free Groq API key at [console.groq.com](https://console.groq.com/signup)
4. Go to Settings → enter your API key
5. Enter topic → click Generate
