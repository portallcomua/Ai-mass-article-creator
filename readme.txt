AI SEO article generator with thematic images, SEO metadata, FAQ Schema, internal linking and WooCommerce licensing readiness.

== Description ==
AI Mass Article Creator helps generate SEO-friendly articles with Groq or OpenAI, thematic images from Pexels/Pixabay/Unsplash/Openverse, meta fields for Rank Math and Yoast, FAQ Schema and internal linking.

== Installation ==
1. Upload the plugin ZIP.
2. Activate it.
3. Add Groq or OpenAI API key.
4. Optional: add Pexels, Pixabay and Unsplash keys for better images.
